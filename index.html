关于 HUB
一个基于区块链的数字内容分发网络，分布式数字内容分发、激励系统。

Hub 是一个为全球数字内容经济服务的区块链价值网络，旨在打造一个高效自治的内容社区，重塑内容创造者、使用者与生态建设者之间的关系，优化现有价值体系。

Hub 是基于委托贡献证明机制（DPOC）的区块链网络，帮助参与者们通过激励机制从中获得应得的收益，让利益回归到价值创造者手中，让价值流动更加透明

Hub 致力于技术创新，帮助更多合作伙伴加速区块链项目落地。提供一系列开源组件，灵活可定制化 SDK。无论是图片、文本还是视频等类型的内容平台均可接入 Hub 网络并自定义激励体系。

我们可以改变什么

为生态参与者提供新的驱动力，改变现有价值分配体系，让所有参与者得到更合理的价值分配

自由定制内容分发方式，提供链上 Bounty Hunters平台，引入第三方合作机构，寻找属于自己最合适的交易方式

提供身份 ID+数字签名链上Hash 存储，链下源文件方式，解决内容版权问题

HUB TOKEN

Hub Token 的愿景是成为以数字内容分发为导向，价值传输为核心的公有区块链网络。Hub Token为 Hub 网络中的唯一价值权益。用户可以通过对社区的贡献获得，也可通过市场交易获得。
